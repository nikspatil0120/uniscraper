  Vertex AI → Gemini API → Groq → Ollama
  (primary)   (fallback)  (rate   (local
                          limit)   backup)

EXTRACTION FLOW:
  1. Try Vertex AI (if enabled & configured)
     ↓ (on error/not configured)
  2. Try Gemini API with retry logic
     ↓ (on 429 rate limit after 2 consecutive failures)
  3. Try Groq (fast cloud, separate quota)
     ↓ (on error)
  4. Try Ollama (local, no quota)
     ↓ (if all fail)
  5. Return regex-extracted data with "failed" marker
